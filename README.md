# Concepção do software
Esse projeto foi pensando para um trabalho acadêmico relacionado ao Tensorflow Keras e redes neurais. Tendo isso em vista, eu idealizei um programa que auxiliasse pessoas comuns e pessoas da área da saúde na identificação de pessoas com a habilidade de concentração debilitada.

Para isso, eu criei um modelo de rede neural próprio com o $\color{blue}{Google \space Teachable \space Machine }$. O modelo conta com 6 classes distintas, que são as seguintes:
- Focado (a) ----> 215 imagens
- Cabeça inclinada ----> 170 imagens
- Sonolento (a) ----> 135 imagens
- Olhando para o lado ----> 286 imagens
- Usando celular ----> 162 imagens
- Ausente
